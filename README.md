# JSON Table Viewer

JSONデータを見やすいテーブル形式で表示するWebアプリケーションです。

## 特徴

- **リアルタイムJSONパース**: JSONを入力すると即座にテーブル形式で表示
- **ソート機能**: カラムヘッダーをクリックして昇順/降順/元の順序でソート
  - 数値列は数値としてソート（文字列として扱わない）
  - 文字列列は辞書順でソート
- **レスポンシブUIレイアウト**:
  - JSON入力エリアとテーブル表示エリアのサイズを自由に調整可能
  - JSON入力エリアの表示/非表示切り替え
- **柔軟なデータ対応**:
  - 配列形式のJSONに対応
  - オブジェクト単体のJSONにも対応
  - ネストされたオブジェクトも文字列化して表示

## 技術スタック

- **Next.js 15** - React フレームワーク
- **React 18** - UIライブラリ
- **TypeScript** - 型安全性
- **Tailwind CSS** - スタイリング

## インストール

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## ビルド

```bash
npm run build
npm start
```

## 使い方

1. 左側のテキストエリアにJSONを貼り付けます
2. 有効なJSONであれば、右側に自動的にテーブル形式で表示されます
3. カラムヘッダーをクリックしてソート（昇順→降順→元の順序）
4. 中央のバーをドラッグして左右のエリアサイズを調整できます
5. 「JSON入力を隠す」ボタンでテーブル表示を全画面表示にできます

### 入力例

```json
[
  {"name": "太郎", "age": 25, "city": "東京"},
  {"name": "花子", "age": 30, "city": "大阪"}
]
```
